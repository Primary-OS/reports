Privacy Policy

Last updated: March 9, 2026

Varick is a Chrome extension that orchestrates multi-LLM research and delivers results into Claude.ai. This policy explains what data the extension accesses, how it is used, and how it is stored.

Data We Collect

Varick does not collect, transmit, or store any data on external servers operated by us. All data remains on your device or is sent directly to third-party AI APIs that you configure.

Data Stored Locally

  • API keys — Your OpenAI, Google Gemini, and xAI API keys are stored in Chrome's local storage on your device. They are never sent to our servers.
  • Pipeline state — Temporary state (current phase, job IDs, research reports) is stored locally while a research pipeline is running and cleared on completion or reset.
  • User preferences — Toggle settings (subagent enable, test mode) are stored locally.

Data Sent to Third Parties

When you run the research pipeline, Varick sends your question and conversation context directly to the following APIs using your own API keys:

  • OpenAI API (api.openai.com) — For o4-mini orchestration, o3-deep-research, and o3 reconciliation
  • Google Gemini API (generativelanguage.googleapis.com) — For Gemini deep-research
  • xAI API (api.x.ai) — For Grok X/Twitter intelligence (optional, only if you provide an xAI key)

These requests go directly from your browser to the respective API endpoints. Varick does not proxy, log, or intercept this traffic.

Website Content Accessed

The extension reads content from the Claude.ai editor (your typed question) and conversation history to provide context to the research pipeline. This content is only used locally and sent to the AI APIs listed above at your request.

Data We Do Not Collect

  • No personally identifiable information
  • No browsing history or web activity
  • No analytics, telemetry, or usage tracking
  • No cookies or cross-site tracking

Data Sharing

We do not sell, transfer, or share any user data with third parties. Your API keys and research content are only transmitted to the AI services you have explicitly configured.

Data Retention

All locally stored data can be cleared at any time via the extension's Settings page ("Clear" button) or by uninstalling the extension. Pipeline state is automatically cleared when a research run completes or is cancelled.

Changes to This Policy

If we update this policy, the revised version will be posted at this URL with an updated date.
